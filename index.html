About

Scientific Calculator is a powerful offline calculator designed for students, teachers, engineers and professionals. It combines advanced scientific calculations with professional matrix operations in a fast, clean and intuitive interface.

Features

Scientific Functions

Trigonometric, logarithmic, exponential, powers, roots and advanced calculations.

Degree & Radian

Instant switching between DEG and RAD modes.

Advanced Matrix Mode

Matrix memory using MatA, MatB and MatC.

Matrix Operations

Addition, subtraction, multiplication, determinant, inverse, transpose and rank.

Editable Expressions

Move the cursor anywhere to quickly correct mistakes.

Offline

No internet connection or account required.
